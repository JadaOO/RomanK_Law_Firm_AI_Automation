# Project for a Phoenix Law Firm to 
- Manage calendar and send push notifications for attorneys
- Manage clients data
- Draft petitions 
- Draft emails
- Handle legal research
- Hanle billings 

## Tools and Languages used 
- Python
- Streamlit
- OpenAI SDK
- RAG
- Pushover
- Sendgrid

## Running the Project
1. git clone the repo 
2. cd to the file
3. source .venv 
4. pip install dependencies
5. streamlit run src/app.py
